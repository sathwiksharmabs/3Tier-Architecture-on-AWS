Built a project **“Highly Available 3-Tier Architecture on AWS”**.  

Used AWS services like VPC, Subnets, Route Tables, Internet Gateway, NAT Gateway, EC2, Auto Scaling Group, ALB, RDS, S3, Route 53, IAM, CloudFront, WAF, ACM, CloudWatch, AWS Backup, and SNS.  

## Key Features

- Designed and implemented a highly available 3-tier architecture using VPC, ALB, Auto Scaling Groups (EC2), and RDS with path-based routing across multiple Availability Zones.  
- Configured Auto Scaling Groups with target tracking policy, along with health checks, CloudWatch alarms, and SNS notifications to ensure automatic scaling and fault tolerance.  
- Deployed an RDS Multi-AZ database with automated backup and recovery, along with cross-region replication to improve data safety and uptime.  
- Secured the infrastructure using IAM roles, security groups, WAF, and private subnet design, and enabled DNS routing through Route 53 and CloudFront with SSL (ACM) for secure content delivery.
